# Move.it 🚀

Aplicação desenvolvida por **Lucas Ferreira Silva** durante a ** #4**,
como parte dos estudos em **Análise e Desenvolvimento de Sistemas**.

---

## 🚀 Tecnologias
Esse projeto foi desenvolvido utilizando:

- React
- Next.js
- TypeScript
- Context API
- Styled Components

---

## 💻 Projeto
O **Move.it** é uma aplicação que utiliza a técnica de *Pomodoro* para incentivar
pausas ativas durante o estudo ou trabalho, promovendo produtividade e bem-estar.

Projeto desenvolvido para fins educacionais e composição de portfólio.

---

## ▶️ Iniciando o projeto
Clone o repositório e acesse a pasta:

```bash
git clone https://github.com/LucasFerreiraProgramador/moveit-next.git
cd moveit-next
```

Instale as dependências:

```bash
yarn
```

Execute a aplicação:

```bash
yarn dev
```
